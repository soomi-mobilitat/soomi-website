# Somi Website

Web oficial de **Somi**, comunitat en moviment.

## Instal·lació

```bash
git clone https://github.com/nom-usuari/somi-website.git
cd somi-website
npm install
```

## Desenvolupament
```bash
npm run dev
```

Obre http://localhost:8080 al navegador.


## Construcció per producció
```bash
npm run build
```
