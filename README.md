# Soomi Website

Web oficial de **Soomi**, comunitat en moviment.

## Instal·lació

```bash
git clone https://github.com/soomi-mobilitat/soomi-website.git
cd soomi-website
npm install
```

## Desenvolupament
```bash
npm run dev
```

Obre http://localhost:8080 al navegador.


## Construcció per producció
```bash
npm run build
```
